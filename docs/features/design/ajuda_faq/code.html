Ajuda / FAQ

Encontre respostas para as suas perguntas mais frequentes.

Como redefinir minha senha?

expand_more

Para redefinir sua senha, clique no link 'Esqueceu a senha?' na página de login. Você receberá um e-mail com instruções para criar uma nova senha. Certifique-se de verificar sua caixa de spam se não receber o e-mail em alguns minutos.

Onde encontro o relatório de atividades mensais?

expand_more

O relatório de atividades mensais pode ser acessado no menu principal, na seção 'Relatórios'. Selecione o período desejado e clique em 'Gerar Relatório' para visualizar ou baixar o documento em formato PDF.

Como cadastrar um novo usuário no sistema?

expand_more

Para cadastrar um novo usuário, vá até a seção 'Usuários' e clique no botão 'Adicionar Novo Usuário'. Preencha os campos obrigatórios, como nome, e-mail e nível de permissão, e salve as alterações. O novo usuário receberá um e-mail para definir sua senha inicial.

O que fazer se o sistema apresentar lentidão?

expand_more

Caso o sistema apresente lentidão, primeiro verifique sua conexão com a internet. Tente limpar o cache do seu navegador ou acessar de uma janela anônima. Se o problema persistir, entre em contato com nosso suporte técnico para que possamos investigar a causa.
